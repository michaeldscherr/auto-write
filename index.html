Dependencies

HTML Markup

Requires jQuery 1.9.1 or later. Add the autowrite.min.js or autowrite.js after your jQuery call directly before the closing body tag.

<script src="assets/js/libraries/jquery.min.js"></script>
<script src="assets/js/autowrite.js"></script>
</body>

Add the data attribute data-role="auto-write" to the direct parent of the children you want to auto write.

<div data-role="auto-write">
	<h1>Welcome to AutoWrite.js</h1>
	<h1>Here you can find documenation.</h1>
	<h1>Have fun and stay frosty.</h1>
</div><!--end auto-write-->

CSS Markup

Add the following markup to your CSS file. Replace the .auto-write class with your own.

The span.line and blink keyframe are only necessary if you would like the cursor to be shown during typing.

.auto-write h1{
	-webkit-opacity: 0;
	-moz-opacity: 0;
	opacity:0;
	text-index:-999999px;
	position: absolute;
}
.auto-write h1.active{
	-webkit-opacity: 1;
	-moz-opacity: 1;
	opacity:1;
	text-indent: 0;
	position: relative;
}
.auto-write span.line{
	color:#de3f29;
	-webkit-animation: blink 1s infinite 0.5s;
	-moz-animation: blink 1s infinite 0.5s;
	animation: blink 1s infinite 0.5s;
}
@keyframes blink{
	0% { opacity:1; }
	50% { opacity:0; }
	100% { opacity:1; }
}
@-webkit-keyframes blink{
	0% { opacity:1; }
	50% { opacity:0; }
	100% { opacity:1; }
}
@-moz-keyframes blink{
	0% { opacity:1; }
	50% { opacity:0; }
	100% { opacity:1; }
}

Default JavaScript Params

var $config = {
	type_duration: 2000,
	still_duration: 2000,
	backspace_duration: 1000,
	human_error_max_duration: 100,
	cursor: true,
	cursor_markup: '<span class="line">|</span>'
};

Param Descriptions

type_duration: how long it takes to type the current text. this will slightly vary depending on the human_error_max_duration, since that time is added to the overall duration of the currently typed text.

still_duration: the timeout between when the text finishes typing and when the backspacing begins. Also the time between backspace completion, and the new text being typed.

backspace_duration: how long it takes to backspace the current text.

human_error_max_duration: max duration (in milliseconds) for the "human typing" effect. Randomly selects a duration between 0 and human_error_max_duration, which is then added to the timeout of the current character being rendered in the typed text.

cursor: boolean to show cursor at end of rendered string while typing.

cursor_markup: markup for cursor if cursor is set to true. Will be set to " " if cursor is false.
